# warOfCards

War of Cards is a simple card game implemented using programming logic to simulate the classic War card game played with a standard deck. The game automatically deals cards between two players and compares them each round to determine the winner.

1. To run our game, you need to make sure you run npm install in the repo folder, after cloning:

```bash
npm install
```

2. After that you can launch the game by running this command

```bash
node game.js
```

## Have fun playing!




## How to play!

### 1. Open the game
In order to initialise the game, type into the terminal 

```bash
node game.js
```

### 2. Rules of the game 

In this game you and the computer have half a deck of cards each. Each player will 'draw' a card and whoever has the higher card, keeps their own card and also gains the other persons card. To win, gain all your oppositions cards.

### 3. Playing the game

After initializing the game. You will see the 'Welcome to' page. To start the game you press "Enter". You will see the first result between you and the computer. Keep pressing "Enter" to get more results and eventually the cards will run out and there will be a Winner and a Loser. Hopefully it's you !!! Good Luck
